DeepL — dedicated machine translation (not an LLM). Excellent quality, but it translates cue by cue without scene context, so idioms/nuance land a touch flatter than an LLM. Free tier = 500k characters/month; the free vs paid endpoint is detected automatically from your key (free keys end in :fx). No model to pick.

Set DeepL as a provider in the priority slots above to use it.

LibreTranslate — open-source machine translation you can self-host (fully local & free, like Ollama but for MT) or point at a public instance. Quality is below the LLMs and DeepL, but it needs no cloud and no key when self-hosted. Like all MT, it translates cue by cue without scene context.

Set LibreTranslate as a provider in the priority slots above to use it.

Google Cloud Translation (official, paid). Create an API key in Google Cloud → APIs & Services → Credentials, with the Translation API enabled.

Microsoft / Azure Translator. Key + the resource region from your Azure Translator resource (Keys and Endpoint page).

Yandex Translate. API key + the folder id of your Yandex Cloud folder.

Cloudflare m2m100 — Cloudflare's dedicated translation model. Reuses the account ID + token from the Cloudflare tab (no separate setup). Free tier, but quality is below the LLMs, and it translates one cue per request (slower on long files).

Set it in the priority slots above. Configure credentials in the Cloudflare tab.

Google Translate (free, no key) — the unofficial keyless endpoint. No sign-up, but quality is poor, it may rate-limit, and it's not meant for heavy use. Here as a last-resort / zero-setup option.

Set it in the priority slots above — no configuration needed.
